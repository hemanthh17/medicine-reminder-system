Using real time we create a medicine reminder system which beeps when the time has been approached
Components required:
Arduino Uno (We can use other Arduino boards also, like Promini, Nano)
RTC DS3231 module
16x2 LCD Display
Buzzer
Led(any color)
Breadboard
Push Buttons
10K Potentiometer
10K,1K Resistors
Jumper Wires
